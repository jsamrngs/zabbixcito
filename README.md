Zabbixcito.

Pequeño sistema de monitoreo inspirado en zabbix, en el puedes monitorear servicios
web y/o dispositivos con ip/mac como impresoras, computadoras, ups entre otros.
Resoponde a los eventos enviando alerta por correo electrónico utilizando smtplib
